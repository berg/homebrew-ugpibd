# homebrew-ugpibd

Homebrew tap for [ugpibd](https://github.com/berg/ugpibd) — a userspace
daemon for USB-GPIB adapters exposing Prologix and HiSLIP TCP front-ends.

## Install

```bash
brew tap berg/ugpibd
brew install ugpibd
```

## Contents

`Formula/ugpibd.rb` is rendered and pushed automatically by ugpibd's release
workflow on each tagged release; it is not meant to be edited here by hand.
